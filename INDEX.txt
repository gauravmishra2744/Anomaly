SYSTEM COMPONENTS
================================================================================

1. LSTM Autoencoder
   - Detects anomalies via reconstruction error
   - 95.18% accuracy

2. Generative AI (VAE)
   - Generates synthetic training data
   - Creates synthetic anomalies

3. Explainable AI (XAI)
   - Explains predictions
   - Feature importance analysis

4. Web Application
   - Flask backend
   - Interactive dashboard
   - REST API
